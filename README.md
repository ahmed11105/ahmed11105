## Featured Projects

### [Dev Quest](https://github.com/ahmedabdurahman/dev-quest)
A gamified full-stack web development bootcamp. 5 levels, 25 coding missions, and 5 boss challenges — all solved in a browser-based code editor. Progress syncs across devices with Supabase.

**Stack:** Next.js, TypeScript, Tailwind CSS, Monaco Editor, Framer Motion, Supabase

### [OSINT Trainer](https://github.com/ahmedabdurahman/social-eng-trainer)
An investigative puzzle game that teaches OSINT and social engineering principles through a realistic Twitter/X clone. Players gather intelligence from profiles, posts, and interactions to crack targets.

**Stack:** Next.js, TypeScript, Tailwind CSS, Framer Motion, Faker.js

### [Signal Lost](https://github.com/ahmedabdurahman/signal-lost)
A mobile mystery app where players investigate a lost signal through a simulated phone interface — contacts, emails, messages, files, and gallery. AI-powered interactions drive the narrative.

**Stack:** Expo, React Native, TypeScript, Zustand, Anthropic AI SDK

### [videogen2](https://github.com/ahmedabdurahman/videogen2)
An AI-powered short-form video generator. Fetches tech news, generates scripts, creates images, synthesizes voiceover, and renders final videos — fully automated.

**Stack:** Next.js, TypeScript, Remotion, MongoDB, OpenAI API, Google Gemini

### [Ali Game: EPIC The Musical](https://github.com/ahmedabdurahman/aliGame)
A saga-themed multiplayer knowledge game for EPIC: The Musical fans. 6 challenge types, streak multipliers, achievements, and a global leaderboard.

**Stack:** Next.js, TypeScript, Zustand, Supabase, Framer Motion, Web Audio API

### [The 1% Club: IntellectGame](https://github.com/ahmedabdurahman/IntellectGame)
A real-time multiplayer logic quiz game. Players join rooms and compete head-to-head with WebSocket-driven game state.

**Stack:** Express.js, Socket.io, Supabase

### [Type Fighter](https://github.com/ahmedabdurahman/type-fighter)
A real-time multiplayer typing competition using peer-to-peer networking. Race against friends with live WPM tracking and sound effects.

**Stack:** Next.js, TypeScript, PeerJS, Tone.js

### [AI Content Engine](https://github.com/ahmedabdurahman/ai-content-engine)
An SEO-optimized content generation and publishing platform with automated blog post creation, tool discovery, and affiliate management.

**Stack:** Next.js, TypeScript, Tailwind CSS, Playwright, Canvas
